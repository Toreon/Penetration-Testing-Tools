Mission

The Pentest Tools Catalogue is a comprehensive reference platform for penetration testing tools and technologies used by Vulnerability Assessment centers, pentesting experts, and related organizations across Europe.

This platform serves as a collective representation of common tools used in the field, building upon recommendations and tools being used by experts throughout Europe.

Curation Principles

Our curation process follows these principles:

  • Relevance: Tools must be actively used in penetration testing or vulnerability assessment
  • Quality: Tools should be well-maintained and have a proven track record
  • Diversity: We aim to represent tools across different categories and use cases
  • Community-driven: Recommendations from European security experts guide our selection

European Expert Contributions

This catalogue is built upon recommendations and tools being used by security experts throughout Europe. We welcome contributions from penetration testers, security researchers, and vulnerability assessment professionals who can help expand and improve this resource.

If you're a security professional with expertise in penetration testing tools, we encourage you to contribute by adding new tools, improving existing entries, or providing feedback on the platform.
